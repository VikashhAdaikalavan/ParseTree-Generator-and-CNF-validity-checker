Detailed Description

Computes truth values of propositional formulas.

Assigns truth values to propositional variables and evaluates the entire formula. Can generate complete truth tables for all possible variable assignments.

Note

Use this constructor when you want to analyze the formula structure before assigning specific truth values, or when generating truth tables.

inline

Collects all unique variables (atoms) present in the parse tree.

Parameters
nodeCurrent node being traversed in the parse tree
atomslistReference to vector that will store unique variable names
Note
This function performs a recursive traversal of the parse tree to identify all leaf nodes (variables). Using a set ensures automatic handling of duplicates with O(log n) insertion time.
The function treats leaf nodes (nodes with no children) as atoms/variables and internal nodes as operators that are not added to the list.

void valuecomputer::computealltruth ( )
inline

Generates and prints complete truth table for the formula.

Iterates through all possible truth value assignments for variables and prints the resulting truth value of the formula for each assignment. Uses binary counting to generate all 2^n combinations where n is the number of variables.
