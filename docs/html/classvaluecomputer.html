Detailed Description

Computes truth values of propositional formulas.

Assigns truth values to propositional variables and evaluates the entire formula. Can generate complete truth tables for all possible variable assignments.

◆ computealltruth()

void valuecomputer::computealltruth ( )
inline

Generates and prints complete truth table for the formula.

Iterates through all possible truth value assignments for variables and prints the resulting truth value of the formula for each assignment. Uses binary counting to generate all 2^n combinations where n is the number of variables.

Evaluation rules:

  • NOT (~): !right
