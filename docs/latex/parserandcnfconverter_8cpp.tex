\doxysubsection{Detailed Description}
Propositional Logic Formula to CNF (Conjunctive Normal Form) Converter. 

\begin{DoxyAuthor}{Author}
K Vikashh Adaikalavan 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025
\end{DoxyDate}
This program converts propositional logic formulas from infix notation to CNF form. It builds parse trees, evaluates truth values, and validates CNF formulas by checking for tautological clauses.

Supported operators\+:
\begin{DoxyItemize}
\item \textquotesingle{}+\textquotesingle{} \+: OR (disjunction)
\item \textquotesingle{}\texorpdfstring{$\ast$}{*}\textquotesingle{} \+: AND (conjunction)
\item \textquotesingle{}\texorpdfstring{$\sim$}{\string~}\textquotesingle{} \+: NOT (negation)
\item \textquotesingle{}\texorpdfstring{$>$}{>}\textquotesingle{} \+: IMPLIES (implication)
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Assumes input formulas are well-\/formed and fully parenthesized 
\end{DoxyNote}

Converts a fully parenthesized infix formula to prefix notation. 

Algorithm\+:
\begin{DoxyEnumerate}
\item Reverse the input string
\item Process each character\+:
\begin{DoxyItemize}
\item \textquotesingle{})\textquotesingle{} → push to stack (becomes opening delimiter after reversal)
\item Operators (+, \texorpdfstring{$\ast$}{*}, \texorpdfstring{$>$}{>}) → push to stack
\item \textquotesingle{}(\textquotesingle{} → pop operators from stack until \textquotesingle{})\textquotesingle{} is found (becomes closing delimiter)
\item Variables/literals → add directly to result
\end{DoxyItemize}
\item Pop any remaining operators from stack
\item Reverse the result to get prefix notation
\end{DoxyEnumerate}

\item Reversing the result to get correct prefix order
