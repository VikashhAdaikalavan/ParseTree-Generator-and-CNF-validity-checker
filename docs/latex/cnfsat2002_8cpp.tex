\doxysubsection{Detailed Description}
CNF (Conjunctive Normal Form) SAT formula validator. 

\begin{DoxyAuthor}{Author}
K Vikashh Adaikalavan 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025

This program reads CNF formulas from a file and validates them by checking if clauses are valid. A clause is considered valid if it contains both a literal and its negation (making it always true).

int main (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Entry point for CNF formula analysis and validation. 

This program iterates through all CNF files in a specified folder, applies CNF validity checking algorithms on each file, and generates an HTML report summarizing the results.

Each file is analyzed for\+:
\begin{DoxyItemize}
\item Validity of the CNF formula (tautological or non-\/tautological)
\item Number of valid and invalid clauses
\item Execution time for the algorithm
\item Memory used during the analysis
\end{DoxyItemize}

The results are presented in an HTML table with color-\/coded rows\+:
\begin{DoxyItemize}
\item {\bfseries{Green}} for valid CNF formulas
\item {\bfseries{Red}} for invalid CNF formulas
\end{DoxyItemize}

\begin{DoxyReturn}{Returns}
Returns 0 on successful execution, -\/1 if the output file cannot be opened.
\end{DoxyReturn}
