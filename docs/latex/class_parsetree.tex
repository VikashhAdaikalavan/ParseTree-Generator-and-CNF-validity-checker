\doxysubsection{Detailed Description}
Builds and manages a parse tree from prefix notation strings. 

Creates a binary tree representation of a logical formula where\+:
\begin{DoxyItemize}
\item Internal nodes are operators
\item Leaf nodes are propositional variables
\item Unary operators (\texorpdfstring{$\sim$}{\string~}) have only right child
\item Binary operators (+, \texorpdfstring{$\ast$}{*}, \texorpdfstring{$>$}{>}) have both children 
\end{DoxyItemize}
