This module handles the conversion of fully parenthesized infix logical formulas to prefix notation for easier parsing and tree construction.

Supported operators\+:
\begin{DoxyItemize}
\item \textquotesingle{}\texorpdfstring{$\sim$}{\string~}\textquotesingle{} \+: NOT (negation) -\/ highest precedence
\item \textquotesingle{}+\textquotesingle{} \+: OR (disjunction) -\/ medium precedence
\item \textquotesingle{}\texorpdfstring{$\ast$}{*}\textquotesingle{} \+: AND (conjunction) -\/ medium precedence
\item \textquotesingle{}\texorpdfstring{$>$}{>}\textquotesingle{} \+: IMPLIES (implication) -\/ lowest precedence
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Assumes input formulas are well-\/formed and fully parenthesized 
\end{DoxyNote}

string infixtoprefix (\begin{DoxyParamCaption}\item[{string}]{s}{}\end{DoxyParamCaption})}



Converts a fully parenthesized infix formula to prefix notation. 

Algorithm\+:
\begin{DoxyEnumerate}
\item Reverse the input string
\item Process each character\+:
\begin{DoxyItemize}
\item \textquotesingle{})\textquotesingle{} → push to stack (becomes opening delimiter after reversal)
\item Operators (+, \texorpdfstring{$\ast$}{*}, \texorpdfstring{$>$}{>}) → push to stack
\item \textquotesingle{}(\textquotesingle{} → pop operators from stack until \textquotesingle{})\textquotesingle{} is found (becomes closing delimiter)
\item Variables/literals → add directly to result
\end{DoxyItemize}
\item Pop any remaining operators from stack
\item Reverse the result to get prefix notation
\end{DoxyEnumerate}

\item Using a stack to handle operator ordering
\item Swapping parentheses roles (due to reversal)
\item Reversing the result to get correct prefix order
\end{DoxyItemize}
\end{DoxyNote}
