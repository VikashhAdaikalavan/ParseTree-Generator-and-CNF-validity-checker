\doxysubsection{Detailed Description}
Represents a node in the parse tree for propositional formulas. 

Each node can represent either\+:
\begin{DoxyItemize}
\item An operator (+, \texorpdfstring{$\ast$}{*}, \texorpdfstring{$>$}{>}, \texorpdfstring{$\sim$}{\string~})
\item A propositional variable (a-\/z, A-\/Z) 
\end{DoxyItemize}
